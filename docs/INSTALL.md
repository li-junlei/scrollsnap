# ScrollSnap 私下分发安装说明

适用版本：`v1.0.0`

## 1. 下载地址

固定下载页：

- https://github.com/li-junlei/scrollsnap/releases

请只从以上 Release 页面下载 APK。

## 2. 安装步骤

1. 在手机设置中允许“安装未知应用”。
2. 下载对应版本 APK（建议选择最新 tag）。
3. 安装后打开应用，根据引导授予权限：
   - 悬浮窗权限
   - 通知权限（Android 13+）
   - Shizuku 权限
4. 先启动 Shizuku，再回到 ScrollSnap 使用。

## 3. 版本升级与回滚

- 每个版本通过 `versionCode` 递增发布，升级直接安装新 APK。
- 如遇问题可回退到 Release 页面上一版本。
- 必须使用同一签名证书发布，才能正常覆盖安装。

## 4. 支持范围

- Android 8.0+（API 26+）
- 需要设备可正常运行 Shizuku
