# ScrollSnap 隐私说明

生效日期：2026-03-05

## 我们收集的数据

ScrollSnap 默认不收集、不上传个人身份信息，不接入广告 SDK，不接入第三方统计 SDK。

## 权限用途

- 悬浮窗权限：用于显示悬浮触发按钮。
- 通知权限：用于前台服务通知展示。
- Shizuku 权限：用于执行截图与自动滑动相关系统命令。

## 本地数据

应用仅在本地保存必要设置（如语言、调试开关、拼接参数、跳过更新版本号）。

## 网络访问

应用会访问 GitHub 接口检查新版本（tag/release 信息），用于更新提示。

## 联系方式

如有隐私相关问题，请通过项目仓库 Issue 联系维护者：

- https://github.com/li-junlei/scrollsnap/issues
